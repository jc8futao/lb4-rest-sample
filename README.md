## Loopback-next rest service example

Getting started

```sh
git clone https://github.com/rahulrkr08/lb4-rest-sample.git
cd lb4-rest-sample
npm start
```

Test your application: 
```sh
curl -X GET http://localhost:3000/rest
curl -X GET http://localhost:3000/rest/1
```
